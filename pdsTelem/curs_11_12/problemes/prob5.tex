  \begin{enumerate}
  \item Trobeu l'expressió per a un sistema de fase mínima $H_{\fm,1}$ i
    un sistema passa tot $H_\pt$ de manera que la composició d'aquests
    ens doni el sistema original.
  \item Trobeu  l'expressió per a un sistema de fase mínima $H_{\fm,2}$ i

  \end{enumerate}
\item 
  Sigui un sistema de fase mínima, amb funció de transferència
  $H_\fm(z)$ tal que
  $$H_\fm(z)H_\pt(z)=H_\fl(z),$$

  passa-tot i un sistema causal de fase lineal generalitzada. Què en
  podem dir dels zeros de $H_\fm$?
