\part{}

El módulo I define los conceptos básicos de la Estadística Descriptiva
y presenta las técnicas básicas para la organización, representación
y análisis de los datos estadísticos.

Las definiciones teóricas se acompañan de múltiples ejemplos para facilitar
la comprensión. Asimismo, se explica cómo resolver los problemas con
la ayuda de aplicaciones informáticas y cómo acceder a bases de datos
estadísticos oficiales.
